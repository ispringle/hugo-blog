The Case of the Wrong Date

I noticed a funny thing happening, the dates for my posts were wrong. The source of the post would contain the correct value in the frontmatter, for example the date of my first post was 2022-07-25 but when that was rendered by NextJS and spit out on the website it was coming in a whole day /earlier_, ie 2022-07-24. At first I thought the culprit was the date library, date-fns, so I dropped into a node shell to see if that was the case...
